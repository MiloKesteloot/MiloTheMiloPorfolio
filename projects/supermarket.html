Supermarket

An optimization puzzle game

Controls

Left Click and Drag to place conveyors or move machines.

Description

Supermarket is a spacial puzzle game based around sorting food items into different areas.

It was originally an experiment to push my level designing skills, but I got good feedback, so I decided to add it to my portfolio!

Notes

Working on this game is when I found out about the PICO-8 character limit! The char limit is what inspired me to start working on my own game engine.
