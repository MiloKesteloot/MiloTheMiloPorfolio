Controls

Use the Arrow Keys to move.

X to restart or select.

Z to open the menu.

Description

Skipmove is a short and challenging puzzle game I created mid-2021.

It was originally an experiment to push my level designing skills, but I got good feedback, so I decided to add it to my portfolio!

Bugs

Currently the only real bug is on the last level (level 15). It's not game-breaking by any means, but it's quite funny. Just look out for it :)
